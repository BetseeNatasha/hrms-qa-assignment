# BUG-001: Dropdown Controls Do Not Open

## Summary

Dropdown fields on the Add Job Advert page do not open or display selectable options.

## Environment

* Application: HRMS Project
* Browser: Google Chrome
* URL: http://localhost:3000/jobadvert/add

## Steps to Reproduce

1. Open the Add Job Advert page.
2. Click Job Position dropdown.
3. Click City dropdown.
4. Click Working Type dropdown.
5. Click Working Time dropdown.

## Expected Result

Dropdown list should appear and allow the user to select a value.

## Actual Result

No dropdown options appear. User cannot select any value.

## Severity

High

## Priority

High

## Impact

Employers cannot complete the Job Advertisement form because required dropdown fields cannot be selected.
